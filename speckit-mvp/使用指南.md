## 🚀 快速开始

### 1. 启动服务

**后端服务 (端口 3001):**
```bash
cd speckit-mvp/backend
npm start
```

**前端服务 (端口 3000):**
```bash
cd speckit-mvp/frontend/public
python3 -m http.server 3000
```

### 2. 访问应用

打开浏览器访问: **http://localhost:3000**

## 🔧 功能说明

### 主要功能
1. **仪表板** - 项目概览和快速操作
2. **项目管理** - 查看、创建、删除项目
3. **搜索功能** - 搜索代码、函数、文档
4. **项目导入** - 从Git仓库或本地文件夹导入项目

### 页面导航
- 点击顶部导航栏的"仪表板"、"项目管理"、"搜索"按钮切换页面
- 使用快捷键 `Ctrl+F` 或 `Cmd+F` 快速进入搜索页面

## 🛠️ 问题排查

### 如果前端页面有报错

1. **检查浏览器控制台**
   - 按 `F12` 打开开发者工具
   - 查看 `Console` 标签页的错误信息
   - 查看 `Network` 标签页的网络请求状态

2. **常见问题及解决方案**

   **问题1: 页面显示"应用初始化失败"**
   - 原因: JavaScript文件加载失败或全局变量未定义
   - 解决: 检查所有JS文件是否正常加载，刷新页面重试

   **问题2: API请求失败**
   - 原因: 后端服务未启动或端口冲突
   - 解决: 确保后端服务在3001端口正常运行

   **问题3: 页面样式异常**
   - 原因: CSS文件加载失败
   - 解决: 检查CSS文件路径，确保文件存在

3. **手动测试步骤**
   ```bash
   # 测试前端页面
   curl http://localhost:3000
   
   # 测试后端API
   curl http://localhost:3001/api/v1/projects
   
   # 运行自动化测试
   node test_integration.js
   ```

- `GET /api/v1/projects` - 获取项目列表
- `POST /api/v1/projects` - 创建新项目
- `GET /api/v1/search` - 搜索功能
- `DELETE /api/v1/projects/:id` - 删除项目

## 🎯 使用建议

1. **首次使用**
   - 建议先导入一个简单的Git仓库进行测试
   - 使用公开的GitHub仓库，如: `https://github.com/octocat/Hello-World.git`

2. **项目导入**
   - 支持Git仓库URL和本地文件夹路径
   - 项目名称会自动从URL或路径生成，也可以手动修改

3. **搜索功能**
   - 支持搜索代码、函数、类、变量等
   - 可以按文件类型筛选搜索结果
